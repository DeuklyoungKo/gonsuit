# gonsuit

**gonsuit.com** — 마이크로 SaaS 허브 사이트

## 기술 스택
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS + shadcn/ui (Maia · Zinc · Indigo)
- **Font**: Inter
- **Deploy**: Vercel

## 로컬 실행

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:3000 접속

## shadcn UI 테마
- **Style**: Maia
- **Base Color**: Zinc
- **Theme**: Indigo
- **Preset URL**: https://ui.shadcn.com/create?preset=ac8UbVQ
