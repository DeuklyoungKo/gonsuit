# gonsuit

**gonsuit.com** — 마이크로 SaaS 허브 사이트

## 🚀 시작하기

이 프로젝트의 메인 소스 코드는 `gonsuit` 디렉토리에 위치해 있습니다.

### 1. 디렉토리 이동
```bash
cd gonsuit
```

### 2. 패키지 설치
```bash
npm install
```

### 3. 로컬 서버 실행
```bash
npm run dev
```

브라우저에서 **[http://localhost:3000](http://localhost:3000)**에 접속하여 사이트를 확인할 수 있습니다.

## 🛠 기술 스택
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS + shadcn/ui
- **Database/Auth**: Supabase
- **UI System**: Maia (Zinc / Indigo)
